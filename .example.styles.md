## Postit

Postit 스타일은 아래와 같이 지정
content는 여러 줄이 가능하며, 각 줄은 | 로 구분함 (즉, |는 문구에 들어가지 못함)
이 내용은 친효 스킨에서 모두 빌려옴
또한, 여러 줄을 쓸 때 `1.`, `-` 등으로 시작하면 ol/ul로 변환하는데 이에 대한 대비는 없음
다음을 사용할 것
• (bullet, U+2022) → 가장 직관적이고 깔끔
‣ (triangular bullet, U+2023) → 조금 더 강조된 느낌
▪ (small square, U+25AA) → 네모 포인트
→ (right arrow) → 진행/순서 강조
⇒ (double arrow) → 강한 강조
➤ (black arrowhead) → 리스트 포인트로 자주 사용
①, ②, ③ … (circled numbers) → 순서 강조, 깔끔한 번호 매김
1), 2), 3) → 괄호 붙이면 마크다운 리스트로 인식되지 않음

https://www.law.go.kr/%ED%96%89%EC%A0%95%EA%B7%9C%EC%B9%99/%ED%95%9C%EA%B8%80%EB%A7%9E%EC%B6%A4%EB%B2%95

〈영화제목, 노래제목〉: U+3008(〈), U+3009(〉) --> Ctrl+Alt+,
《책제목》: U+300A(《), U+300B(》) --> Ctrl+Alt+.
작은 따옴표: U+2018(‘), U+2019(’) --> Ctrl+Alt+'
큰 따옴표: U+201C(“), U+201D(”) --> Ctrl+Alt+Shift+'

␣(visible space) U+2423 --> Ctrl+Alt+Space

## mathjax

아래 사이트에서 쉽게 작성 가능
https://latexeditor.lagrida.com/

(inline math)
$$\frac{2S}{3} = 2a + 2b + 2c + d + e$$

(block math, inline에서도 물론 사용 가능)
$$\begin{align}
\frac{2S}{3} & = 2a + 2b + 2c + d + e\\
\frac{S}{3} & = a + b + c + 2d
\end{align}$$

(align* 도 가능함. 단, 이것도 서로 매치되어야 함)
$$\begin{align*}
i+m-1 & \le n-l-1\\
